Facebook Graph API Integration

This dashboard is a simulation for demo purposes. To connect with real Facebook comments, integrate with the Facebook Graph API.

1 Create a Facebook App

Go to Facebook Developers and create a new app.

  • Select "Business" as the app type
  • Add "Pages Public Content Access" and "Pages Messaging"
  • Get your App ID and App Secret

2 Get Page Access Token

Generate a Page Access Token with permissions:

pages_read_engagement, pages_manage_posts, pages_read_user_content

3 Webhook Setup

Set up a webhook for real-time comment notifications:

app.post('/webhook/facebook', (req, res) => {
  const { entry } = req.body;
  entry.forEach(pageEntry => {
    pageEntry.changes.forEach(change => {
      if (change.value.item === 'comment') {
        processComment(change.value);
      }
    });
  });
  res.sendStatus(200);
});

4 Fetch & Reply

Use these Graph API endpoints:

GET /{post-id}/comments?access_token={TOKEN}

POST /{comment-id}/comments
  ?message={AI_REPLY}&access_token={TOKEN}

Important Notes

  • Facebook requires Business Verification for some permissions
  • Rate limits: 200 calls/hour per user for comments
  • Always handle errors gracefully and implement retry logic
  • Store tokens securely (environment variables only)
  • Use Redis/Bull message queue for high-volume pages
